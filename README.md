## Deploy to GitHub Pages
1. Create a repo named `portfolio-joseph-caballero` on GitHub.
2. Push this project to that repo.
3. Run:
```bash
npm run deploy
```

Vite is configured with `base: '/portfolio-joseph-caballero/'` for GH Pages.
